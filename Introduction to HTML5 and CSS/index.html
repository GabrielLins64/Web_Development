Página de Introdução ao HTML 5

Esta página e as páginas referenciadas - "linkadas" têm o intuito de utilizar o máximo possível de ferramentas de HTML 5, CSS, JavaScript e PHP para fins didáticos e de reutilização de código. Para visualizar o código fonte das páginas pressione Ctrl + U (para Firefox e Google Chrome). Nos comentários é possível encontrar mais informações acerca das ferramentas utilizadas.
