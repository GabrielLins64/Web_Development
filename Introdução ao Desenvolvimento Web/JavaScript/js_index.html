JavaScript

JavaScript é uma linguagem de programação interpretada (o código fonte é diretamente executado por um interpretador), estruturada (paradigma de programação que enfoca na sequência, decisão e iteração) e de alto nível (com a sintaxe mais próxima da linguagem humana). É uma das principais ferramentas do desenvolvimento Web, e permite que páginas e aplicações Web sejam interativas.

Alguns exemplos da potencialidade do JavaScript, são: A funcionalidade do acesso rápido deste site; a animação de "brilhar" o div ao ser selecionado; a modificação e inserção dinâmica de textos e botões em cada página; etc. Se quiser, visualize aqui o código JavaScript importado por esta página.

Inserindo na página

Para inserir o JavaScript numa página HTML, você pode importá-lo de um arquivo com a extensão ".js" utilizando a tag: <script src="Caminho/Ate/O/arquivo.js"></script>. Ou então, utilize a tag <script> e insira seu código javascript aqui dentro </script>.

Para fazer um pequeno teste, insira o código ao lado em uma página HTML e execute para ver o resultado. Se desejar visualizar logo os efeitos do código nesta página, clique no botão "executar".
<script>
 alert("Hello World!");
</script>

Variáveis

Por padrões de comunidade, escreve-se os identificadores em JavaScript com camelCase. Para se declarar uma variável em JavaScript, pode-se usar um destes 3 tipos de palavras reservadas:

  1. var: Declara uma variável.
  2. let: Declara uma variável local (escopo de bloco)
  3. const*: Declara uma constante (utilizada apenas para leitura)
    4. *Constantes em JS não podem ser reatribuídas, no entanto, objetos constantes podem ter suas propriedades modificadas.

Exemplo: var identificadorDaVariavel = 32;

Tipos de Dados

Na versão mais recente do JavaScript, há 7 tipos de dados. Os primitivos: Booleans, null, undefined, Number (engloba inteiros, ponto flutuantes e dupla precisão), String e Symbol (suas instâncias são imutáveis). E o tipo Object (o qual pode ser instanciado sem necessariamente pertencer a uma classe).

Arrays

Arrays em JavaScript são declarados de forma similar ao Python (com exceção da palavra reservada de variável no início). Arrays possuem vários métodos embutidos que facilitam sua manipulação, você pode visualizá-los neste link.
Ex.: var identificadorDoArray = ["Dado 1", 64, true];

No código ao lado vamos criar alguns tipos de dados em variáveis e mandar um "alert" printar um array que irá conter tais variáveis.
<script>
 var variavel = "String";
 const numero = 100;
 let bool = true;
 let array = [variavel, numero, bool];
 alert(array);
</script>

Document Object Model (DOM)

O Document Object Model (DOM) é uma API (application programming interface) para documentos HTML e XML. É uma representação em objetos da página web, que pode ser manipulada através da utilização de linguagens de programação, como o JavaScript.

Um exemplo de utilização da API DOM, é o uso do objeto document para obter, criar ou alterar elementos (e suas propriedades) da página. Concretizando ainda mais, como exemplo podemos modificar o HTML e o CSS de algum elemento da página, buscando-o pelo seu id (ou de outras maneiras).

Observe o código ao lado e faça seus experimentos em sua página teste.
<script>
 var botao = document.getElementById("botao_executar");
 botao.onclick = function() {
  botao.innerHTML = "Executado!";
  botao.style.color = "red";
 };
</script>
