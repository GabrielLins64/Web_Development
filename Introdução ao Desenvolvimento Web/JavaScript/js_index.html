JavaScript

JavaScript é uma linguagem de programação interpretada (o código fonte é diretamente executado por um interpretador), estruturada (paradigma de programação que enfoca na sequência, decisão e iteração) e de alto nível (com a sintaxe mais próxima da linguagem humana). É uma das principais ferramentas do desenvolvimento Web, e permite que páginas e aplicações Web sejam interativas.

Alguns exemplos da potencialidade do JavaScript, são: A funcionalidade do acesso rápido deste site; a animação de "brilhar" o div ao ser selecionado; a modificação e inserção dinâmica de textos e botões em cada página; etc. Se quiser, visualize aqui o código JavaScript importado por esta página.

Inserindo na página

Para inserir o JavaScript numa página HTML, você pode importá-lo de um arquivo com a extensão ".js" utilizando a tag: <script src="Caminho/Ate/O/arquivo.js"></script>. Ou então, utilize a tag <script> e insira seu código javascript aqui dentro </script>.

Para fazer um pequeno teste, insira o código ao lado em uma página HTML e execute para ver o resultado. Se desejar os efeitos do código nesta página, clique no botão "executar".
<script>
 alert("Hello World!");
</script>

Document Object Model (DOM)

O Document Object Model (DOM) é uma API (application programming interface) para documentos HTML e XML. É uma representação em objetos da página web, que pode ser manipulada através da utilização de linguagens de programação, como o JavaScript.

Um exemplo de utilização da API DOM, é o uso do objeto document para obter, criar ou alterar elementos (e suas propriedades) da página. Concretizando ainda mais, como exemplo podemos modificar o HTML e o CSS de algum elemento da página, buscando-o pelo seu id (ou de outras maneiras).

Observe o código ao lado e faça seus experimentos em sua página teste.
<script>
 var botao = document.getElementById("botao_executar_2");
 botao.onclick = function() {
  botao.innerHTML = "Executado!";
  botao.style.color = "red";
 };
</script>
