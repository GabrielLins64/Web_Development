HTML

HTML (HyperText Markup Language -- Linguagem de Marcação de Hipertexto) é uma linguagem de *marcação (não de programação!) utilizada para construir páginas Web. Ao escrever um documento em HTML, os softwares navegadores, como Firefox, Google Chrome, Safari, etc., interpretam o que está escrito e traduzem em elementos visuais para o usuário.
Linguagem de marcação: sistema de escrita de textos a ser interpretado de modo que seja sintaticamente diferenciável.

Esta página mostrará elementos de HTML e de CSS, bem como o resultado da aplicação destes. No entanto, maior parte do código só será exibido ao analisar o código fonte da página. Pode-se visualizar o mesmo de duas formas:

2. Clique com o botão direito em alguma sessão específica que deseja ver o código e clique em "Inspecionar elemento".

CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets ou Folhas de Estilo Cascata) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML. Isto é, adiciona estilo (cores, fontes, posicionamento, espaçamento, efeitos, etc.) a um documento web. Existem 3 maneiras de adicionar estilo a um documento Web com CSS:

 1. Aplicando diretamente na tag, com a propriedade style:
<span style="color: red; font-weight: bold"> Texto vermelho e negrito </span> ;
 2. Utilizando a tag <style> e inserindo o css dentro da tag;
 3. Importando o estilo de um arquivo ".css", utilizando a tag:
<link rel="stylesheet" href="arquivocss.css">.

Tags

Tags são elementos que definem um tipo de container com suas propriedades específicas (que podem ser editadas com CSS). Tags são inseridas, no código fonte da página, iniciando-se com um símbolo de menor que "<", seguido do nome da tag e finalizando com o símbolo de maior que >, ficando: <nome_da_tag>. Por exemplo, temos as tags <html>, <p>, <body> que representam, respectivamente, o início de um documento HTML (uma pagina web), um parágrafo, e o corpo da pagina. As tags terminam com os mesmos identificadores, mas inserindo-se uma barra "/" antes do nome da tag.
Exemplo: <tag> conteúdo </tag>

Todo o conteúdo e outras tags que estiverem entre o início e término das tags, são parte interna destas e estão sujeitos às propriedades de estilo desta. Para descobrir mais tags e obter mais informações sobre estas, visualize o código fonte desta página.

Estrutura da página

A estrutura básica do código fonte de uma página HTML 5, tem o formato exibido abaixo:

<!-- Tudo entre estes identificadores são comentários (não influenciam a página) -->
<!-- O comando abaixo não é uma tag, ele informa ao navegador que é um documento do tipo HTML 5 -->
<!DOCTYPE html>

<!-- A tag abaixo "html" é o container para todos os outros elementos HTML. Ela declara o
início da descrição de uma página. O atributo lang declara e informa aos mecanismos de busca e aos
navegadores qual a linguagem utilizada. -->
<html lang="pt-br">

<!-- A tag head é um container para meta dados (definições como título, conjunto de caracteres,
estilos (css) e scripts (javascript)). -->
<head>

 <!-- A tag title especifica o título da página, que é mostrado na aba do navegador.
 O título é de suma importância para os mecanismos de busca. -->
 <title>Título da página</title>

</head>

<!-- A tag body define o corpo do documento e comporta todos os elementos necessários à página. -->
<body>

 <!-- A tag h1 é um cabeçalho de importância 1 (maior importância) -->
 <h1>Cabeçalho</h1>

 <!-- A tag p define um parágrafo, e contém algumas propriedades de margem e indentação. -->
 <p>Parágrafo</p>

</body>

</html>

Coloque o código acima num editor de texto de sua preferência, salve com o formato ".html" e abra a página num navegador (na maioria dos sistemas operacionais é só abrir o arquivo).

Containers

Em HTML, containers são áreas delimitadas por tags, e são utilizados para organização e posicionamento de elementos internos (incluindo outros containers) e de si mesmos na página. Um parágrafo é um container, que contém suas propriedades de estilo, bem como a "caixa preta" que você visualiza cada seção desta página. Textos e Parágrafos

Em HTML os textos podem ser inseridos sem nenhuma tag ou dentro de tags de formatação de texto como parágrafos, seções, span, entre outros. Textos puros (sem estarem dentro de uma tag de formatação) não possuem margem ou padding (ver a sessão de CSS) definidos, sua indentação e espaçamentos entre as linhas superiores e abaixo são definidos pelas propriedades de estilo internas do container onde o texto se localiza. As quebras de linha são inseridas automaticamente ao chegar no final do container, ou podem ser inseridas manualmente utilizando-se a tag <br>. Observe o exemplo abaixo:

Texto puro, sem nenhuma tag. Quebra de linha: <br>
Nova linha, após a tag "break line" (br).

<p> Texto com parágrafo. (Observe as margens acima e abaixo) </p>

Após o parágrafo (sem tag de formatação). Após este ponto amarelo, foi dada uma quebra de linha no editor (veja o código fonte), mas não é visualizada no navegador, pois não chegou no final do container nem foi forçada uma quebra de linha.

Imagens

Para adicionar uma imagem à página, utilize a tag img. Por exemplo:
<img src="Imagens/matrixcode.jpg" width="600" height="300" alt="matrix">

Para adicionar uma imagem como plano de fundo (da página, ou de qualquer container),
deve-se utilizar o css: background-image: url("caminho/até/a/imagem.jpg");

Links

Links (referências) para outras páginas podem ser inseridos com a tag <a>seguida do nome que deseja que apareça, conforme o exemplo abaixo:

<a href="https://www.google.com.br/" target="_blank"> Google </a><br>
*O atributo (target = "_blank") faz com que a página seja aberta numa nova aba."

Tabelas

Para definir uma tabela em HTML, utiliza-se a tag <table>. Para definir uma linha, utiliza-se <tr>, para células em negrito (de cabeçalho) a tag <th> e, por fim, para as células normais, a tag <td>. É possível fazer com que uma célula ocupe 2 ou mais linhas, bem como 2 ou mais colunas, utilizando os atributos "rowspan" e "colspan", respectivamente. Para mais detalhes, visualize o código fonte da tabela abaixo:

3,0 3,1 3,2 3,3

Símbolos ASCII

Ao escrever os textos com conteúdo de uma página, as vezes é necessário utilizar alguns símbolos ASCII como "<", "®", ou "ß". No entanto, além de os símbolos "<" e "&" serem caracteres reservados para o html (ao inserí-los no código fonte da página, o browser interpreta como um comando, não como um símbolo de texto), as vezes é difícil encontrar o símbolo para inserir na página. Portanto, alguns sites fornecem uma tabela com o código HTML para inserir, em formato de texto, o símbolo. Por exemplo, para inserir na página o símbolo <, devemos colocar, no código fonte, a sequência &#x3C. No link abaixo é possível visualizar uma tabela com alguns destes.
